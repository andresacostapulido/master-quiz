# Master Quiz - Sistema de Cuestionarios Interactivos

Sistema de quiz interactivo para preparación de exámenes del Magister en DevOps.

## 🎯 Características

- **Múltiples asignaturas**: SecDevOps y Administración de Sistemas Cloud
- **Modos de estudio**: Normal (todos los temas) y Repaso por tema específico
- **Autenticación**: Login con Google para guardar progreso
- **Estadísticas avanzadas**: Heatmap, curva de aprendizaje, predicción de aprobación
- **Respuestas trampa**: Preguntas diseñadas con opciones similares para mayor desafío

## 📚 Asignaturas

- **SecDevOps**: 30 preguntas, 5 temas
- **Admin Sistemas Cloud**: 36 preguntas, 10 temas

## 🚀 Despliegue en GitHub Pages

1. **Sube el código:**
   ```bash
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/TU_USUARIO/master-quiz.git
   git branch -M main
   git push -u origin main
   ```

2. **Activa GitHub Pages:**
   - Settings → Pages
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save

3. **Accede:** `https://tu-usuario.github.io/master-quiz`

## 🔒 Nota sobre Firebase

Las credenciales de Firebase client-side son seguras de exponer públicamente. Firebase protege tus datos mediante reglas de seguridad en la base de datos, no ocultando las credenciales.
